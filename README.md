# ことばカード

えいごも まなべる 知育フラッシュカード（無料・広告なし・登録不要）

お子さま向けに、日本語と英語のことばを絵文字カードで楽しく学べるWebアプリです。

## 特長

- **140枚以上のカード** — どうぶつ・うみのいきもの・むし・たべもの・のりもの・いろ・かず・かたち・しぜん・からだ の10カテゴリ
- **音声読み上げ** — 日本語と英語をネイティブ音声で再生（端末内蔵の読み上げ機能を使用、通信不要）
- **フラッシュモード** — 自動再生で次々にカードをめくって読み上げ
- **クイズモード** — 「これはなにかな？」3択クイズ。正解するとピンポーン♪
- **PWA対応** — ホーム画面に追加すればアプリのように使える・オフラインでも動く
- **完全無料** — 広告なし・課金なし・個人情報の収集なし

## 技術構成

- 素のHTML/CSS/JavaScript（ビルド不要・依存ゼロ）
- 画像はすべて絵文字（ライセンスの心配なし）
- 音声は Web Speech API（TTS費用ゼロ）
- 効果音は WebAudio で生成(音声ファイルなし)
- Vercel等の静的ホスティングにそのままデプロイ可能

## ローカルで動かす

```bash
python3 -m http.server 4173 --directory .
# → http://localhost:4173 を開く
```

※ 音声読み上げ・オフライン機能(Service Worker)は https 環境または localhost で動作します。
